Importent FAQ

What is the DOM in JavaScript?

The Document Object Model (DOM) is a programming interface that allows programmers to manipulate the structure, style, and content of a web page

How do you find an element in the DOM?

The easiest way to access a single element in the DOM is by its unique ID. We can get an element by ID with the getElementById() method of the document object.

What is an event delegation model, and why is is useful?

Event Delegation is a pattern based upon the concept of Event Bubbling. It is an event-handling pattern that allows you to handle events at a higher level in the DOM tree other than the level where the event was first received.
1. Attaching event listeners to each child element can be inefficient, especially when you have many elements or dynamically generated elements.
2. With event delegation, you only need to attach one event listener to the parent element, reducing the memory footprint and improving performance.

How to manipulate attributes in JavaScript?

1. hasAttribute(name) – to check for existence.
2. getAttribute(name) – to get the value.
3. setAttribute(name, value) – to set the value.
4. removeAttribute(name) – to remove the attribute.
5. attributes is a collection of all attributes.
